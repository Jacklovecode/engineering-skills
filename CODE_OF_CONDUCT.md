# 社区行为准则

## 我们的承诺

我们希望这个项目成为一个尊重、务实、适合学习和协作的工程社区。

参与者应当：

- 保持尊重和建设性。
- 讨论想法、设计和代码时，避免攻击个人。
- 清楚说明取舍和理由。
- 接纳不同经验水平的贡献者。
- 让讨论聚焦在项目改进上。

## 不可接受的行为

不可接受的行为包括：

- 骚扰、侮辱或人身攻击。
- 歧视性语言或行为。
- 未经允许公开他人的私人信息。
- 反复偏离技术讨论并干扰协作。
- 提交恶意代码、敏感信息或可能造成误解的署名。

## 执行方式

当行为伤害项目或贡献者时，维护者可以删除评论、关闭 Issue、拒绝 PR，或限制相关人员继续参与。

如果你发现问题，请优先通过私下渠道联系维护者。若没有私下渠道，可以创建 GitHub Issue，但请只公开安全的信息，不要暴露隐私或敏感细节。
