# 变更记录

本项目遵循语义化版本思路记录重要变更。

## 0.1.0

初始公开版本。

### 新增

- 13 个核心技能：`start`、`zoom-out`、`deepen`、`clarify`、`grill`、`plan`、`execute`、`tdd`、`diagnose`、`review`、`finish`、`git`、`skill-edit`。
- Codex、Claude、Gemini 和通用智能体入口说明。
- 安装、结构校验、压力场景验证和技能辅助脚本。
- GitHub Issue 模板、PR 模板、贡献指南、社区行为准则和安全策略。

### 说明

- 技能正文以中文为主，技能名和平台清单保持英文，便于多平台识别。
